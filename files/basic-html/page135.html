                                    附录 II ：渔具与渔获图鉴一、传统渔具1、蚬篮：这种篮原本是在海滩上洗蚬仔（也叫土鬼，海瓜子）用的，所以叫蚬篮。2、筒网示意图，学名腿缯网，也叫七星网，潮间双桩有翼定置张网。3、徛桁网，也叫企桁网，学名樯张网。外港流水急风浪大，前后各须八至九条的张索绑定。4、木帆牵缯，木帆牵缯是传统的拖网作业方式，依靠风帆动力。渔网上绠浮子是用桐油煮过杉木浮子 ( 类似现在抽纸形状的扁长方形 ,另一种约 20 公分长中有孔的橄榄形 ) 或玻璃浮球，沉子为圆形指环状瓷子。渔获以带鱼、红娘、赤鲫、肉鱼、午笋等为主，是风帆时代的代表。5、机帆拖网，机帆拖网是渔业机械化的标志。渔获保鲜改用碎冰制冷。6、大围缯是大型围网，用于捕捞大黄鱼、带鱼等。单网高捕获量。过度捕捞导致大黄鱼资源急剧衰退。130
